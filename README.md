# Colorsmosis

Colorsmosis is a color picking tool that you can use to build color palettes with, from an image of your choosing. Choose the number of colors you want to pick from your image, drag pins around your image to pick colors, and save their hex codes for later.

It's built with Astro + React, and react-draggable https://github.com/react-grid-layout/react-draggable

Hosted on netlify at https://colorsmosis.netlify.app/

Code Architecture + Docs are inside the src/ folder
